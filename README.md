# Shreyas P Lokhande — Portfolio

Personal portfolio website built as a single self-contained HTML file.

## 🚀 Live Site
https://shree1314.github.io/my-portfolio/

## 📁 Structure
```
index.html   ← entire site (HTML + CSS + JS + images all embedded)
README.md    ← this file
```

## 🌐 Deployment
Hosted on GitHub Pages from the `main` branch root (`/`).

## 🎨 Tech
- Vanilla HTML / CSS / JavaScript
- Google Fonts (Playfair Display + DM Sans)
- All images base64-embedded — no external dependencies
